## Скрипты для настройки нового устройства на Android

Здесь я максимально отключаю приложения, которые безопасно отключать. Те, что нужны, я возвращаю через установку из Google Play.

### Используйте на свой страх и риск.
Если что-то пошло не по плану, то можно сбросить настройки до заводских через меню или Recovery.

## Пояснение, как это работает

Системные приложения в Android полностью удалить нельзя без root-прав. Но можно их "заморозить", так чтобы они лежали как файлы, но при этом никак не запускались при старте системы. Делается это через приложение ADB и режим отладки на смартфоне.

Подробнее про ADB/Fastboot можно узнать на форуме:  
[https://4pda.to/forum/index.php?showtopic=383300](https://4pda.to/forum/index.php?showtopic=383300)

Скрипт написан специально для Linux, и в системе должны быть установлены утилиты ADB/Fastboot. Об этом тоже можно узнать на форуме 4pda.

На смартфоне нужно включить режим отладки по USB в разделе "Для разработчиков". Тогда будут доступны команды ADB через USB-кабель. Например:

```bash
adb devices
```

Выведет доступные устройства, если в смартфоне включен режим отладки.

Вообще, утилита ADB позволяет делать многое со смартфоном через компьютер. Например, можно запустить быструю установку группы приложений или удалить группы приложений.

Теперь, когда у нас доступна отладка через USB и установлены adb-tools, можно запускать скрипт по "заморозке" приложений. Список приложений, которые будут заморожены, находится в `apps_list.txt` — в него можно добавлять свои или удалять те, которые не нужно отключать.

```bash
./debloat.sh
```

Можно наблюдать в реальном времени, как множество приложений исчезает из списка меню в смартфоне.
